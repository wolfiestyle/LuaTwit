Functions

request (method, url, body, headers, filter)
Performs an HTTP request.

Parameters:

  • method HTTP method.
  • url Request URL.
  • body Post body. Form-encoded string (single part) or table (multipart).
  • headers Additional headers.
  • filter Function to be called on the result data.

Returns:

  1. Response body or nil on error.
  2. Status code.
  3. Response headers.

Class future

Future object. It contains the result of an asynchronous HTTP request.
future:peek (no_upd)
Checks (non-blocking) and returns the value of the future if it's ready.

Parameters:

  • no_upd Don't ask the service for new data, just return the current stored value (default false).

Returns:

  1. true if the value is ready, otherwise false.
  2. List of return values from the HTTP request.
future:wait ()
Waits (blocks) until the value of the future is ready.

Returns:

    List of return values from the HTTP request.
future:cancel ()
Cancels the HTTP request associated with this object.

Returns:

    nil if successfully cancelled, or the result values if called after the request finished.

Class stream

Stream object. It represents an open connection that continuously receives data.
stream:is_active (no_upd)
Checks if the stream connection is open.

Parameters:

  • no_upd Don't ask the service for new data, just use the current stored state (default false).

Returns:

    true if the connection is open, or false and the response if it's closed.
stream:next (no_upd)
Returns the next object in the stream queue.

Parameters:

  • no_upd Don't ask the service for new data, just use the current stored state (default false).

Returns:

    A stream object, or nil if there is no more data available.
stream:iter ()
Convenience function for iterating the stream in a for loop.

Returns:

    Stream iterator.
stream:close ()
Closes the stream connection.

Returns:

    HTTP request result.

See also:

    future:cancel

Class service

HTTP service object. Executes HTTP requests on background (with curl.multi).
service.new ()
Creates a new async HTTP client.

Returns:

    New instance of the async client.
service:set_conn_limits (total_conn, host_conn)
Sets the connection limits.

Parameters:

  • total_conn Maximum number of total connections (default unlimited).
  • host_conn Maximum number of connections per host (default unlimited).
service:update ()
Performs data transfers and checks for finished requests.

Returns:

    If there was data available, the current number of unfinished requests. Otherwise nil.
service:wait (timeout)
Waits until data is received.

Parameters:

  • timeout Time to wait, in milliseconds (default 1000).

Returns:

    Number of objects with data to be read.
service:http_request (method, url, body, headers, filter, is_stream)
Performs an asynchronous HTTP request.

Parameters:

  • method HTTP method.
  • url Request URL.
  • body Post body. Form-encoded string (single part) or table (multipart).
  • headers Additional headers.
  • filter Function to be called on the result data.
  • is_stream Indicates if it's a streaming connection or not.

Returns:

    future object with the result.
