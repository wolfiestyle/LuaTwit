Class future

Future object. It contains the result of an asynchronous API call.
future:peek ()
Checks (non-blocking) and returns the value of the future if it's ready.

Returns:

  1. true if the value is ready, otherwise false.
  2. List of return values from the API call.
future:wait ()
Waits (blocks) until the value of the future is ready.

Returns:

    List of return values from the API call.

Class service

OAuth service object. Executes OAuth requests on background threads.
service.new (keys, endp, threads)
Creates a new async OAuth client.

Parameters:

  • keys Table with the OAuth keys (consumer_key, consumer_secret, oauth_token, oauth_token_secret).
  • endp Table with OAuth endpoints.
  • threads Number of threads to create (default 1).

Returns:

    New instance of the oauth async client.
service:start ()
Starts the worker threads.
service:stop ()
Stops the worker threads.
service:data_available ()
Checks if there is data pending to be received.

Returns:

    true if there is data pending on the message queue, otherwise false.
service:call_method (name, callback, ...)
Performs an asynchronous OAuth request.

Parameters:

  • name Method name.
  • callback Function that processes the response output (called when reading the future).
  • ... Method arguments.

Returns:

    future object with the result.
service:PerformRequest (method, url, request, headers, callback)
Async wrapper for OAuth.PerformRequest.

Parameters:

  • method HTTP method.
  • url Request URL.
  • request Table with request pairs.
  • headers Custom HTTP headers.
  • callback Function that processes the response output.

Returns:

    future object with the result.
