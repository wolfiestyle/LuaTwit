Functions

type (obj)
Gets the type of the supplied object.

Parameters:

  • obj Any value.

Returns:

    The _type field if it's a table. If not present or not a table, the Lua type.
check_args (args, rules, r_name)
Checks if the arguments in the specified table match the rules.

Parameters:

  • args Table with arguments to be checked.
  • rules Rules to check against.
  • r_name Name of the resource that is being checked (for error messages).

Returns:

  1. The args table with the values coerced to their types, or nil on error.
  2. The error string if args is invalid.
resource_call (res, client, args)
Performs an API call with the data on a resource object.

Parameters:

  • res Resource object (from luatwit.resources).
  • client API client (luatwit.api instance).
  • args Table with method arguments.

Returns:

    The result of the API call.

See also:

    raw_call
object_call (obj, args)
Performs an API call with the data from an object returned by other API calls.

Parameters:

  • obj Table returned by luatwit.api.raw_call.
  • args Table with method arguments.

Returns:

    The result of the API call.
resource_builder (method, path)
Creates a resource builder object. It's used to construct the fields of luatwit.resources.

Parameters:

  • method HTTP method.
  • path Resource path.

Returns:

    Resource builder object. Must call :finish() after the construction is done.
read_config (filename)
Reads a simple key = value style config file.

Parameters:

  • filename File to be read.

Returns:

    Table with the config values.
